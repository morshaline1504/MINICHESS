ℹ About MiniChess

MiniChess is a variant of chess played on a 6×5 board with standard chess pieces and rules.

Board Setup (6 rows × 5 columns):

  • Black: Rook, Knight, Queen, King, Bishop + Pawns
  • White: Rook, Knight, Queen, King, Bishop + Pawns

Features:

  • Minimax algorithm with alpha-beta pruning
  • Check and checkmate detection
  • Advanced position evaluation
  • Multiple difficulty levels (Depth 1-4)
  • Three game modes with reset option
  • Move validation and legal move filtering

Rules:

  • King capture ends the game
  • Check must be resolved
  • Checkmate = Winner declared
  • Stalemate = Draw
  • Pawns promote to Queen on last rank
